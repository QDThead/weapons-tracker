### 2. Production Verification Sub-Tab (Major Feature)

**New Supply Chain sub-tab (#13):** "Verification"
- 18 cobalt facility card grid (mines + refineries)
- **Sentinel-2 satellite thumbnails** (120x120) next to overlay charts — click to open 512x512 zoom modal
- **Overlay chart** per facility: red thermal FRP bars + purple NO2 ratio bars + white dashed capacity reference line (Chart.js, dual Y axes)
- **Cloud coverage indicator** — warning banner when <50% clear, scoring adjustment prevents false discrepancy flags from cloud cover
- **Verification score (0-100%):** compares satellite activity against reported production status
  - CONSISTENT (≥80%) — satellite matches claims
  - INCONCLUSIVE (50-79%) — partial data or heavy cloud cover
  - DISCREPANCY (<50%) — satellite contradicts claims
- Sort: Score ↑↓, Name, Country, Capacity
- Filter: All / DISCREPANCY / INCONCLUSIVE / CONSISTENT
- "REPORTED PAUSED" label for facilities with pause/suspension notes
