### 4. Live Cobalt Forecasting Engine

**New file: `src/analysis/cobalt_forecasting.py`** (303 lines)

Replaces hardcoded placeholder data with live computed forecasts:

| Forecast | Computation | Data Source |
|----------|-------------|-------------|
| **Price** | Fetch FRED nickel monthly → apply 2.0x cobalt/nickel ratio → quarterly averages → linear regression → 4 quarters projected | FRED `PNICKUSDM` (live, free) |
| **Lead Time** | Base transit days + chokepoint risk delays (7/4/1 days by severity) + disruption probability (25% of base for critical routes) | Existing shipping route data |
| **Insolvency** | Taxonomy financial (60%) + economic (40%) scores → quadratic scaling to 0-60% probability → adjusted by Altman Z-Score | Taxonomy scores + dossier data |
| **Supply Adequacy** | From existing sufficiency scenario data | Sufficiency model |
| **Signals** | Auto-generated from all computed data + top risk factors | All of the above |

**New API endpoint:** `GET /globe/minerals/{name}/forecast`
